QT4CG Specifications

This page lists the specifications published from the qtspecs repository of the XSLT Extensions community group.

The latest drafts of

  • XSLT Transformations (XSLT) Version 4.0
  • XPath and XQuery Functions and Operators 4.0
  • XML Path Language (XPath) 4.0, and
  • XQuery 4.0: An XML Query Language

are updated automatically when changes are made to the repository.
