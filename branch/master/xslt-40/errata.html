Abstract

This document records all known errors in the XSL Transformations (XSLT) Version 2.0 specification; for updates see the latest version of that document.

The errata are numbered, classified as Substantive or Editorial, and listed in reverse chronological order of their date of publication in each category. Each entry contains the following information:

  • The section in which the error exists.

  • A description of the error.

  • A correction to the error, if applicable.

Changes to the text of the spec are indicated thus: new, added text, modified text, and . Substantive corrections are proposed by the XSL Working Group (part of the XML Activity), which have consensus that they are appropriate; they are not to be considered normative until approved by a Call for Review of Proposed Corrections or a Call for Review of an Edited Recommendation.

Please send comments about this document to W3C XSLT/XPath/XQuery public comments mailing list, public-qt-comments@w3.org. It will be very helpful if you include the string [XSLTerrata] in the subject line of your comment. An archive of that mailing list is available at http://lists.w3.org/Archives/Public/public-qt-comments/.
