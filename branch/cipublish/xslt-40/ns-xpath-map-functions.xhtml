1 Introduction

This document describes the namespace http://www.w3.org/2005/xpath-functions/map defined by the [XSLT 3.0] specification (November 2015 version) as well as the [FO 3.1] specification (December 2014 version). This namespace is identified by the namespace prefix map:, which is not a predefined prefix. For updated information, please refer to the latest version of the [XSLT 3.0] and [FO 3.1] specifications.

This document describes the names that are defined in this namespace at the time of publication. The W3C reserves the right to define additional names in this namespace in the future. [XSLT 3.0] and [FO 3.1] are the only specifications that may amend this namespace, and the intention is to keep them aligned, though there may be timing differences between publications.

This document contains a directory of links to related resources, using RDDL (as defined in [Resource Directory Description Language (RDDL)]).

It is GRDDL-enabled (as defined in [Gleaning Resource Descriptions from Dialects of Languages (GRDDL)]), that is to say that a GRDDL-compliant processor can extract useful RDF (as defined in [Resource Description Framework (RDF): Concepts and Abstract Syntax]) representations of the information contained herein.

2 Map Functions

This section lists all of the functions defined by the [XSLT 3.0] specification to be in the namespace reserved for certain functions operating on maps. Each function is uniquely identified with a URI of the form: “http://www.w3.org/2005/xpath-functions/map#name” where name is the name of a function, such as “get”: http://www.w3.org/2005/xpath-functions/map#get.

The normative definitions of these functions are in the [XSLT 3.0] specification. For convenience, a very brief, non-normative summary of each function is provided. For details, follow the link on the “Summary:” introductory text below each function.

contains

xs:boolean contains(map(*), xs:anyAtomicType)

Summary:

Tests whether a supplied map contains an entry for a given key

entry

map(*) entry(xs:anyAtomicType, item()*)

Summary:

Returns a map that contains a single entry (a key-value pair).

for-each

item()* for-each(map(*), function(xs:anyAtomicType, item()*))

Summary:

Applies a supplied function to every entry in a map, returning the concatenation of the results.

get

item()* get(map(*), xs:anyAtomicType)

Summary:

Returns the value associated with a supplied key in a given map.

keys

xs:anyAtomicType* keys(map(*))

Summary:

Returns a sequence containing all the keys present in a map

merge

map(*) merge(map(*)*)

Summary:

Returns a map that combines the entries from a number of existing maps.

put

map(*) put(map(*), xs:anyAtomicType, item()*)

Summary:

Returns a map containing all the contents of the supplied map, but with an additional entry, which replaces any existing entry for the same key.

remove

map(*) remove(map(*), xs:anyAtomicType)

Summary:

Returns a map containing all the entries from a supplied map, except (typically) one.

size

xs:integer size(map(*))

Summary:

Returns the number of entries in the supplied map.
